## Website Report

### 1. Structure of a Webpage


A webpage is built using **HTML** (HyperText Markup Language). Every HTML document begins with a `<!DOCTYPE html>` declaration, which tells the browser what type of document it is reading.

The document is then divided into two main sections inside the `<html>` tag:

- **The `<head>` section:** This is not visible on the page itself. it has information *about* the page, such as the `<title>` (which appears in the browser tab), links to stylesheets, and meta information for search engines.
- **The `<body>` section:** This section contains all the visible content: headings, paragraphs, images, lists, links, and everything visitors will see and interact with in the browser..

HTML uses tags that usually come in pairs: an opening tag like <p> and a closing tag like </p>, with the content placed between them. Some tags, such as <br/> for a line break and <hr/> for a horizontal rule, are self-closing and do not require a closing tag.

Colours on a webpage use the **RGB (Red, Green, Blue)** colour system. Each colour is made by mixing different amounts of red, green, and blue light. In HTML, colours can be written using *RGB values*, such as `rgb(255, 209, 220)` for peach or `rgb(243, 166, 219)` for pink. Opacity can also be added using RGBA, for example `rgba(74, 134, 200, 0.6)`.

### 2. Features of a Good Website

A good website should have the following features:

- **Clear layout and design:** The page should be easy to read, with text and images neatly arranged, and the background colour should complement the text rather than clash with it.
- **Easy navigation:** 
All internal and external links should work correctly; broken links make the site look unreliable and frustrate visitors.
- **Working links:** All internal and external links should work properly. Dead links are a sign of poor quality and frustrate users.
- **Appropriate use of images:** Images should be sized appropriately and include ALT text for accessibility, because too many large pictures can slow the website down.
- **Consistent style:** Use the same fonts, colours, and layout across all pages so the site feels consistent and professional.
- **Readable text:** Text size, font choice, and colour should make reading comfortable, and avoid all caps or too many different styles because they make text harder to read.
